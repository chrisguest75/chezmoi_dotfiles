## Steps

1) Install `homebrew` [Instructions Here](https://brew.sh/
) and [Configuration Here](https://docs.brew.sh/Homebrew-on-Linux)  
1) Install `chezmoi` [Instructions Here](https://www.chezmoi.io/install/)  


```sh
cd .local/share/chezmoi
# or
chezmoi cd

# debug issues
chezmoi doctor


```

## Adding files

```sh
chezmoi --debug diff

```
